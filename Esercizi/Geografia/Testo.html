Geografia

Realizzare un sistema che memorizzi una mappa geografica mondiale. Tutte le classi sono nel package mondo .

R1: Stati

Ogni stato è identificato da diversi attributi, tra cui la città capitale, gli stati confinanti, le lingue parlate, la popolazione, la dimensione in km-quadri:.
  • Il metodo aggiungiStato(String stato, String capitale, long popolazione, double dimensione ) della classe Mondo aggiunge uno stato al sistema.
  • Il metodo getCapitale(String stato) ritorna una stringa con il nome della capitale.
  • Il metodo getPopolazione(String stato) e getDimensione(String stato) ritornano rispettivamente un long e un double.
  • Il metodo setConfine(String stato1, String stato2) stabilisce che 2 stati confinano.

R2: Lettura da File

        Il metodo caricaDati(String fileStati, String fileConfini) carica in memoria la struttura dati da due file. 
  • Il file stati.txt contiene le informazioni su ogni singolo stato in una linea nel seguente formato "Stato ; capitale dimensione popolazione", ad esempio:
            Italy :    Rome     116,305     58,133,509
            Jamaica :    Kingston     4,244     2,758,124
            Japan :    Tokyo     145,882     127,463,611

        La dimensione è in kmq, la popolazione in unità (dati ufficiali del 2007)   
        Il file usa notazione anglosassone, quindi le cifre decimali sono separate dal carattere '.' mentre la virgola ',' è usata come separatore di migliaia.
        Possono esserci un numero arbitrario di spazi tra i dati su una stessa linea.
  • Il file confini.txt contiene le informazioni sui confini; ogni linea contiene una coppaid istati confinanti separati da ":", ad esempio:
            Spain : Portugal
            Spain : France
            Italy :Switzerland
            Italy : France
            Slovenia:  Italy
            ......   
           

R3: Confini

  • Il metodo confinano(String stato1, String stato 2) ritorna true se i 2 stati confinano false altrimenti.
  • Il metodo numeroConfinanti(String stato) dato il nome dello stato ritorna un int contenente il numero di stati confinanti.
  • Il metodo getConfinanti(String stato) dato il nome dello stato ritorna una ritorna l'elenco degli stati confinanti separati da virgola; ad esempio, supponendo di aver caricato la mappa europea completa,  getConfinanti("Italy") ritorna "France, Switzerland, Austria, Slovenia, Vatican, San Marino": l'ordine non conta.
R4: Statistiche

Il sistema deve permettere di calcolare alcune statistiche.
  • Il metodo ordinaPerPopolazione() ritorna una stringa contenente la lista degli stati in ordine decrescente di popolazione. Ogni linea della lista contiene il numero di posizione, il nome, il numero di abitanti, nel seguente formato "1. China  1321851888" .
  • Il metodo ordinaPerDimensione() ritorna una stringa contenente la lista degli stati in ordine decrescente di dimensione. Ogni linea della lista contiene il numero di posizione, il nome, la dimensione in kmq, nel seguente formato "1. Russia  6592735" .
  • Se la lista è vuota ritorna la stringa vuota "".
